cgpVcf

cgpVcf contains a set of common perl utilities for generating consistent Vcf headers and an installation method which resolves some upatched issues in the Tabix package (reported).

It primarily exists to prevent code duplication between some other projects.

Dependencies/Install

Some of the code included in this package has dependencies on several C packages:

  • Tabix
  • vcftools

And various perl modules.

Please use setup.sh to install the dependencies. Please be aware that this expects basic C compilation libraries and tools to be available, most are listed in INSTALL.
